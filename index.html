SolidWorks Standard Primitives

Set of few helpful macros

swsp is a set of few standard primitive macros, aimed at shortening the time required to create new part in SolidWorks, by automating some repetitive tasks.

For more information please check out the README.

Requirements

SWSP requires .NET 3.5 Framework to run, and, obviously, SolidWorks.

The tool was developed & tested with SolidWorks 2013 & 2014.
