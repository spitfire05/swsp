SolidWorks Standard Primitives

Set of few helpful macros

Status: WIP

This is a toolset of few macros that help to speed-up the creation process of basic of common shapes. Currently it is still very "Work In Progress"

Download

Latest release

Requirements

SWSP requires .NET 3.5 Framework to run, and, obviously, SolidWorks.

The tool was developed & tested with SolidWorks 2013 & 2014.
